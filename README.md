# Random Quote Machine

This is the first of the Front End challenges from freeCodeCamp.com

It is a website built with React.js that deliveres a random nice quote every time the user press a button. The web is full responsible and the quote can be shared in Twitter through a link.

## Getting Started

Should you want to run the app, download the package and open the project with an IDE like Visual Studio Code. Then, you need to have installed npm or yarn in your machine. If you already have one of these, write npm start or yarn start in the command line. It will automatically run the App in the port 3000 of your localhost.
